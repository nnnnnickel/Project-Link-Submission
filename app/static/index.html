欢迎使用 Agentic Tarot

一个由 Agent 驱动的塔罗解读小工具，先看看怎么用：

  • 💬
    提出问题
    在左侧「问题」框里写下你想咨询的事情。
  • 🔮
    抽牌 或 输入牌组
    可以点「抽一张牌」连续抽取（抽几张解读几张），也可以切换到「输入牌组」手动填写已有的牌面。
  • 🔁
    新解读 / 追问
    第一次提问用「新解读」；得到解读后可以切到「追问」继续对话，Agent 会记得上下文。
  • 🚪
    退出
    聊完之后点「退出」结束本次会话、清空记忆，页面会自动尝试关闭；如果浏览器拦截了自动关闭，请手动关闭该页面。
